# Rock Genres — 摇滚子流派编曲特征

## Classic Rock (经典摇滚)

| 维度 | 特征 |
|------|------|
| **节奏** | 4/4 拍，中速 100-130 BPM，shuffle 或直拍 |
| **吉他** | 过载音色，open chord + power chord 混用 |
| **贝斯** | 旋律性 walking bass line，不只是根音 |
| **鼓** | 律动感强，镲片多用于点缀 |
| **曲式** | 传统结构，solo 段必备 |
| **代表作** | Led Zeppelin, The Rolling Stones, AC/DC |

---

## Hard Rock (硬摇滚)

| 维度 | 特征 |
|------|------|
| **节奏** | 120-150 BPM，强力的 down picking |
| **吉他** | 高增益失真，riff 驱动，drop D 常见 |
| **贝斯** | 厚重、与底鼓紧密结合 |
| **鼓** | 军鼓重击 2&4，crash 密集 |
| **曲式** | 大段 chorus，solo 通常较旋律化 |
| **代表作** | Deep Purple, Guns N' Roses, Aerosmith |

---

## Punk Rock (朋克摇滚)

| 维度 | 特征 |
|------|------|
| **节奏** | 快 160-200 BPM，全十六分镲片 |
| **吉他** | 小功率失真，down stroke，极少 solo |
| **贝斯** | 简单根音八分音符 |
| **鼓** | 简单暴力，少 fill，crash 乱砸 |
| **曲式** | 极简：2-3和弦，Verse-Chorus，通常<3分钟 |
| **代表作** | Ramones, Sex Pistols, Green Day |

---

## Alternative Rock (另类摇滚)

| 维度 | 特征 |
|------|------|
| **节奏** | 多变 80-140 BPM，常变拍子 |
| **吉他** | 清音 + 失真对比，chorus/delay 效果丰富 |
| **贝斯** | 旋律性更强，偶做主旋律 |
| **鼓** | 动态大，从安静到爆发的反差 |
| **曲式** | 非传统结构，intro 长或 outro 实验性强 |
| **代表作** | Radiohead, Nirvana, Pixies |

---

## Metal (金属)

| 维度 | 特征 |
|------|------|
| **节奏** | 140-240 BPM，双踩十六分或三连音 |
| **吉他** | 极高增益，palm mute，tremolo picking |
| **贝斯** | 跟随节奏吉他 riff 或独立对位 |
| **鼓** | 双底鼓/双踩，blast beat 可能 |
| **曲式** | 复杂 riff 变化，solo 技术要求高 |
| **代表子类** | Thrash, Heavy, Death, Power, Progressive Metal |

---

## Grunge (垃圾摇滚)

| 维度 | 特征 |
|------|------|
| **节奏** | 中慢速 80-120 BPM，大量 half-time feel |
| **吉他** | 厚重失真 + 清音交替，fuzz 踏板标志性 |
| **贝斯** | 厚重 muddy 音色，常走独立线条 |
| **鼓** | 松散、自然的律动，少 trigger/量化 |
| **曲式** | Verse-Chorus 为主，动态对比极大(quiet→LOUD) |
| **代表作** | Nirvana, Pearl Jam, Soundgarden |

---

## Indie Rock (独立摇滚)

| 维度 | 特征 |
|------|------|
| **节奏** | 中速 100-140 BPM，常有不规则节奏 |
| **吉他** | jangly clean tone，arpeggio 常见 |
| **贝斯** | 旋律驱动，比传统摇滚更突出 |
| **鼓** | 简洁有力，lo-fi 质感可接受 |
| **曲式** | 自由，bridge/pre-chorus 灵活运用 |
| **代表作** | Arctic Monkeys, The Strokes, Vampire Weekend |

---

## Progressive Rock (前卫摇滚)

| 维度 | 特征 |
|------|------|
| **节奏** | 复杂多变，5/4, 7/8, 9/8 等非常见拍号 |
| **吉他** | 技巧性强，大量 clean arpeggio + 失真 solo |
| **键盘** | 重要角色！synth, organ, piano 频繁使用 |
| **贝斯** | 技术性 walking line，与鼓对话 |
| **曲式** | 超长曲式 (8-20min)，多段落拼接，主题重现 |
| **代表作** | Pink Floyd, Yes, Rush, Tool |

---

## 风格融合建议

混合流派时注意平衡：

- **Punk + Metal** = Crossover Thrash / Metalcore（保留朋克速度 + 金属 riff）
- **Alternative + Classic Rock** = 现代摇滚主流（另类编曲 + 传统 solo）
- **Grunge + Prog** = 重型前卫（大动态 + 复杂拍号）
- **Indie + Classic** = 复古独立（jangly 吉他 + 传统曲式）
