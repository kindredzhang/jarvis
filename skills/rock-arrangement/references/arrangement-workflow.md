# Arrangement Workflow — 编曲详细工作流程

## 流程总览

```
Motive/Riff → 曲式框架 → 鼓编配 → 贝斯线 → 节奏吉他 → 主音吉他/键盘 → 人声旋律定位 → 过渡细化 → 动态平衡检查
```

---

## 一、从动机到曲式

### 如果你有一个 riff
1. 确定 riff 的和声进行（几级和弦）
2. 判断 riff 适合当 verse 还是 chorus 的素材
   - 能量高、有记忆点 → chorus
   - 铺垫性强、有空间 → verse
3. 设计对比段落：若 riff=verse，写一个更开放的 chorus

### 如果你有一个和声进行
1. 确定调性和节奏
2. 设计节奏吉他 pattern（扫弦 vs 分解）
3. 为每个段落设计不同的 voicing（不同把位/转位）

### 如果你从零开始
1. **定调**：E, A, D, G 是摇滚最常用的调（方便吉他）
2. **定速**：根据情绪选择 BPM
   - 80-100：沉重/groove
   - 100-130：经典摇滚
   - 130-180：朋克/金属
   - 180+：极端金属
3. **定风格**：参考 `rock-genres.md`

## 二、段落织体设计技巧

### 从 Verse 到 Chorus 的织体变化

| 元素 | Verse (薄) | → | Chorus (厚) |
|------|-----------|:-:|-----------|
| 鼓 | Hi-Hat + Snare + Kick | → | 加 Crash + 更多桶鼓 |
| 贝斯 | 简单根音 | → | 八度跳或更密集节奏 |
| 节奏吉他 | Palm mute / 分解和弦 | → | 开放扫弦 / power chord |
| 主音吉他 | 无或点缀 | → | 加入 riff/riff 变体 |
| 人声 | 较低音域，单轨 | → | 高音域，双轨+伴唱 |

### Pre-Chorus 的桥梁作用
- 长度：4 小节
- 鼓：军鼓滚奏或 open hi-hat 增加紧张感
- 和声：五度进行（V → I）或上升半音级进
- 目的：告诉听众"chorus 要来了"

### Bridge 的转折
- 调性变化：走向 IV、VI、bVII 等
- 织体削减：至少去掉一种乐器
- 节奏变化：可切换到 half-time 或 double-time

## 三、常见编曲套路

### "Quiet-Loud" 动态 (Grunge/Alternative)
```
Verse (清音/分解) → Chorus (全失真爆发)
```

### "Wall of Sound" 音墙 (Shoegaze/Heavy)
```
多轨吉他叠失真 + 混响，所有乐器同时强奏
```

### "Call & Response" 问答 (Classic Rock/Punk)
```
吉他 riff → 人声回应该 riff 的节奏
```

### "Stop & Go" 停顿 (Funk Rock/Nu Metal)
```
乐队全停 → 仅 riff 独奏 → 全队爆发
```

## 四、编配检查清单

编曲完成后逐项检查：

- [ ] 每个段落都有独特的织体特征？
- [ ] Verse 到 Chorus 有明显的动态/能量提升？
- [ ] 所有过渡点都有 fill 或标记变化？
- [ ] 乐器间无频段冲突（低频、中频、高频都有人负责）？
- [ ] Solo 段有充分的背景支撑（鼓加密度，贝斯稳根音）？
- [ ] Outro 有明确的结束方案（渐弱/强收/回旋重复）？
- [ ] 整首歌长度合理（3-5分钟为佳，prog 除外）？

## 五、制作阶段建议

| 阶段 | 要点 |
|------|------|
| **录音前** | 用参考曲 (reference track) 对标音色和编曲 |
| **录音** | 节奏吉他双轨录左右 pan，增加宽度 |
| **混音** | 先平衡音量，再 EQ 清理，最后压缩粘合 |
| **Mastering** | 摇滚通常需要较响的 RMS (-8 到 -12 dB)，注意不要压死动态 |
