# Rock Arrangement — 摇滚乐编曲制作

## Overview

为摇滚乐提供从灵感雏形到完整编曲的全流程指导。涵盖乐器配置、段落结构、动态设计、过渡处理、风格化制作。适用于 classic rock、hard rock、punk、alternative、metal、grunge、indie rock 等。

## 核心原则

| 原则 | 说明 |
|------|------|
| **律动优先** | 鼓与贝斯的 groove 是摇滚的骨架，先定节奏再铺和声 |
| **织体分层** | 用节奏/主音/和弦吉他 + 键盘 构建厚度层次 |
| **动态对比** | 段落间力度、密度、音域的反差越大，听感越强 |
| **"少即是多"** | 每个乐器都有"呼吸空间"，避免频段打架 |

## 工作流程

### 1. 需求诊断

先确认用户当前状态：

- **有完整 demo？** → 优化编曲，解决具体问题
- **有动机/ riff？** → 围绕它构建完整编曲
- **从零开始？** → 定风格 → 定调/速度 → 设计曲式

### 2. 曲式设计 (Song Structure)

按需选择或组合：

| 段落 | 作用 | 典型长度 |
|------|------|----------|
| Intro | 建立氛围/主题动机 | 4-8 小节 |
| Verse | 叙事/铺垫，织体较薄 | 8-16 小节 |
| Chorus | 高潮记忆点，织体最厚 | 8-16 小节 |
| Bridge | 情感转折/调性变化 | 8 小节 |
| Solo | 器乐展示，情绪顶点 | 8-16 小节 |
| Outro | 渐弱/fade out/强收 | 4-8 小节 |

常见结构：`Intro → Verse → Chorus → Verse → Chorus → Bridge → Solo → Chorus → Outro`

### 3. 乐器编配

参见 `references/instrumentation.md` 获取各乐器详细编配指南。

**基本配置清单：**

- **鼓**：定义节奏型和填充点
- **贝斯**：与底鼓配合，构建律动线
- **节奏吉他**：和声铺底（开放/封闭和弦，或 power chord）
- **主音吉他**：riff、licks、solo 填充
- **主唱**：旋律线定位
- **伴唱**：和声填充 chorus/bridge

### 4. 过渡与动态

| 过渡类型 | 做法 |
|----------|------|
| **build-up** | 军鼓滚奏 + 吊镲渐强 + 吉他反馈音 |
| **breakdown** | 去除大部分乐器，只留 riff 或鼓 |
| **half-time feel** | 速度感觉减半，加重低音冲击 |
| **fill in** | 鼓 fill + 吉他滑音引导段落切换 |

### 5. 风格化参考

参见 `references/rock-genres.md` 获取各子流派的具体编曲特征。

## 输出格式

为用户产出一个清晰的 **编曲方案**，包含：

```
🎸 [歌曲名称] — 编曲方案

风格：xxx
调性：xxx | BPM：xxx

【曲式结构】
[段落] [小节数] [织体描述]

【乐器编配】
- 鼓：
- 贝斯：
- 吉他 1 (节奏)：
- 吉他 2 (主音)：
- 键盘(如有)：
- 人声：

【关键过渡点】
[小节] → [过渡手法]

【制作建议】
- 音色/效果器推荐
- 混音频段注意点
```

## 使用时机

- 用户说 **"帮我编一首摇滚"** / **"摇滚编曲"** / **"写段摇滚"**
- 用户有 riff 需要扩展成完整编曲
- 用户想优化现有编曲的结构和动态
- 用户需要特定风格（punk / metal / alternative 等）的编曲建议
